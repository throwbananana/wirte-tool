# 基调大纲模块需求说明

## 一、功能目标

在现有“大纲图模块”中，新增一个**基调大纲模块**，用于以可视化曲线方式展示：

- 整体情节推进趋势
- 多个人物在同一时间轴下的情绪 / 立场 / 能量 / 参与度变化
- 人物与情节之间的对应关系
- 人物与人物之间在关键节点上的压制、和解、对立、互动方向

该模块核心是：  
以一条**围绕 x 轴上下波动的情节线**作为主基调线，其余**人物基调线**围绕该基准线展开，支持用户对每个波动节点进行编辑、增删、引入、收束、关联和说明。

---

## 二、模块组成

### 1. 基调图区域
用于展示所有曲线和节点关系。

包含：

- **x轴（平铺线）**
  - 表示时间线 / 剧情推进线 / 章节顺序
- **情节线**
  - 一条围绕 x 轴上下波动的主曲线
  - 表示整体情节强弱、冲突高低、节奏起伏
- **人物基调线**
  - 多条围绕平铺线上下波动的曲线
  - 每条线对应一个人物
  - 用于表示人物在剧情中的状态、参与强度、情绪倾向、立场变化等

### 2. 节点编辑区域
用于编辑每个曲线节点的信息。

每个节点支持：

- 调整位置
- 调整波动幅度
- 调整曲率
- 添加说明
- 添加关联箭头
- 在该节点引入 / 减少 / 收束人物线

### 3. 汇总模块
用于将所有基调线、节点、关系进行分类汇总展示。

---

## 三、基调图核心规则

### 3.1 情节线

情节线为全局主线，规则如下：

- 围绕 x 轴上下波动
- 每个波峰 / 波谷 / 转折点都为可编辑节点
- 节点可表示：
  - 冲突增强
  - 情绪反转
  - 节奏加速
  - 信息揭露
  - 事件爆发
  - 阶段收束

用户可控制：

- 节点位置
- 节点强度
- 节点曲率
- 节点前后过渡平滑度
- 节点说明文字

### 3.2 人物基调线

人物线围绕平铺线波动，但受情节线影响，可与情节线形成对应关系。

每条人物线支持：

- 新增人物线
- 删除人物线
- 从某个节点开始引入人物线
- 在某个节点结束 / 收束人物线
- 在某个阶段降低存在感
- 在某个节点重新强化人物线

人物线节点支持：

- 上下拖拽，表示强度变化
- 左右拖拽，调整变化发生时机
- 调整节点曲率
- 在节点上添加人物说明
- 与其他线段建立关系箭头

---

## 四、节点能力定义

每个曲线节点都应具备以下属性：

### 4.1 基础属性
- 节点名称
- 所属线
- 节点时间位置
- 节点强度值
- 节点类型
- 节点说明

### 4.2 节点类型
建议支持以下类型：

- 普通节点
- 高潮节点
- 转折节点
- 引入节点
- 收束节点
- 冲突节点
- 和解节点
- 对应节点

### 4.3 节点操作
支持：

- 新增节点
- 删除节点
- 拖拽节点
- 调整曲率
- 添加备注
- 复制节点样式
- 与其他节点建立关系

---

## 五、人物线生命周期操作

人物线需要支持“从某节点开始”和“到某节点结束”的过程控制。

### 5.1 引入人物线
用户可以在任意节点新增人物线，表示：

- 人物首次登场
- 人物开始介入主线
- 人物被提及但未完全展开

### 5.2 减少人物线
表示：

- 人物弱化
- 人物退场
- 人物暂时离开主线
- 人物影响力下降

### 5.3 收束人物线
表示：

- 人物阶段性完成
- 角色弧线闭合
- 与主线关系结束
- 当前阶段不再参与

### 5.4 再引入人物线
收束后可再次从后续节点重新引入，形成阶段性回归。

---

## 六、说明信息添加规则

支持在以下位置添加文字说明：

### 6.1 全过程说明
可在一段曲线区间上添加说明，描述：

- 本阶段人物状态
- 本阶段情节功能
- 该段关系变化
- 该段冲突趋势

### 6.2 顶部节点说明
可在波峰 / 波谷 / 关键转折点添加说明，描述：

- 人物心理变化
- 情节事件说明
- 冲突爆发原因
- 关系转折点
- 伏笔 / 回收点

### 6.3 说明类型
建议区分：

- 人物说明
- 情节说明
- 关系说明
- 伏笔说明
- 结论说明

---

## 七、汇总模块需求

汇总模块用于将所有线条、节点、关系整理成结构化视图。

### 7.1 按基调线汇总
系统根据所有基调线列出：

- 情节线明细
- 人物线明细
- 每条线对应的全部节点
- 每个节点的强度值
- 节点说明
- 节点关系

### 7.2 节点强度汇总
展示每个节点的强度信息，例如：

- 时间点
- 所属线
- 节点强度
- 节点类型
- 是否为关键节点

可用于快速识别：

- 高强度冲突点
- 多人物交汇点
- 关键剧情节点

### 7.3 平级节点汇总
识别同一时间线上的多个节点，形成“平级节点组”。

例如同一个时间点可能同时存在：

- 人物A节点
- 人物B节点
- 情节线节点

系统需要展示：

- 同时间点涉及哪些人物
- 人物与情节的对应关系
- 同时发生的冲突 / 和解 / 转折

### 7.4 对应关系汇总
汇总显示：

- 某人物在哪些情节节点有响应
- 某情节节点关联了哪些人物
- 某人物与其他人物在哪些节点发生关系变化

---

## 八、关系箭头功能

在曲线节点处，可以新增一条**垂线箭头**，指向同一时间节点的其他线段，用于表示人物与人物、人物与情节之间的关系。

### 8.1 箭头创建规则

用户可在某节点执行：

- 拖拽生成垂直关系线
- 指向同个时间节点上的其他人物线 / 情节线节点
- 一个节点可建立多个关系箭头

### 8.2 箭头类型

支持以下类型：

#### 1）实线单箭头
- 含义：压制
- 方向：箭头指向被动方
- 说明：主动方对被动方形成明显控制、施压、主导

#### 2）虚线单箭头
- 含义：和解
- 方向：箭头指向接受方 / 被影响方
- 说明：主动方发起缓和、沟通、修复

#### 3）实线对立双箭头
- 含义：强对抗
- 双向均有对立
- 说明：双方处于明显冲突和压制状态

#### 4）虚线相向双箭头
- 含义：双向和解 / 靠近
- 说明：双方都在靠拢、理解、修复关系

#### 5）实线相反双箭头
- 含义：相互压制 / 拉扯
- 说明：双方都有对抗、争夺主导权

#### 6）虚线相反双箭头
- 含义：表面疏离但存在缓和互动
- 说明：方向相反表示关系未完全一致，虚线表示冲突减弱或复杂过渡

### 8.3 线型语义规则

#### 实线
表示：
- 压制
- 强冲突
- 强控制
- 直接作用

#### 虚线
表示：
- 和解
- 缓和
- 间接影响
- 柔性连接

#### 箭头方向
用于表示：
- 主动 / 被动
- 施加 / 接受
- 发起 / 响应

---

## 九、建议交互设计

### 9.1 曲线编辑
支持：

- 点击新增节点
- 拖拽调整节点位置
- 拖拽控制贝塞尔曲线曲率
- 框选多个节点批量调整
- 吸附同一时间线节点

### 9.2 人物线操作
支持：

- 从工具栏新增人物线
- 拖动线头接入某节点开始
- 拖动线尾在某节点收束
- 颜色区分不同人物
- 显示 / 隐藏某人物线

### 9.3 关系箭头操作
支持：

- 从节点向下或向上拖出关系箭头
- 自动吸附同时间点的其他节点
- 通过右键菜单切换箭头类型
- 显示关系说明标签

### 9.4 说明编辑
支持：

- 节点旁气泡注释
- 悬浮显示摘要
- 点击展开完整说明
- 支持人物说明 / 情节说明分类

## 十、建议数据结构

可以按以下对象模型设计：

### 10.1 线对象 Line
- id
- name
- type（plot / character）
- characterId
- color
- startNodeId
- endNodeId
- visible
- description

### 10.2 节点对象 Node
- id
- lineId
- timeIndex
- intensity
- curveIn
- curveOut
- nodeType
- title
- description
- tags

### 10.3 关系对象 Relation
- id
- fromNodeId
- toNodeId
- relationType
- lineStyle（solid / dashed）
- arrowMode（single / double-opposed / double-facing / double-reverse）
- meaning
- note

### 10.4 汇总对象 SummaryGroup
- timeIndex
- relatedNodes[]
- relatedCharacters[]
- plotNodeId
- intensityLevel
- relationCount

## 十一、汇总模块展示形式建议

建议提供三种视图切换：

### 1）按人物看
展示某个人物全程节点、强度变化、涉及情节节点、关系变化

### 2）按情节看
展示情节线每个关键节点对应了哪些人物、关系和说明

### 3）按时间看
展示同一时间点上所有平级节点的对应关系

---

## 十二、可直接作为原型页面的模块划分

### 页面结构建议

#### 左侧
- 人物列表
- 线条管理区
- 节点类型工具栏
- 箭头关系工具栏

#### 中间
- 基调图主画布
- x轴时间线
- 情节线 / 人物线编辑区

#### 右侧
- 节点详情面板
- 关系详情面板
- 说明编辑面板

#### 底部 / 独立页签
- 汇总模块
- 节点明细表
- 平级节点表
- 关系表

---

## 十三、一句产品定义

**基调大纲模块**是一种围绕时间轴展示“情节起伏、人物波动、节点关系与多线对应”的可视化大纲工具，用于帮助用户构建复杂叙事中的节奏、人物和关系结构。
